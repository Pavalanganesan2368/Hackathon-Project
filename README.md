# Legacy Website UI Modernization
This Project modernizes a legacy website using HTML, CSS and Javascript.

## Project Overview
This project is developed as part of a hackathon to solve the problem of modernizing legacy website UI.

## Problem Statement
Many legacy websites have outdated UI which affects user experience and accessibility. The goal is to create a new UI website with modern standards.

## Solution
I'm Create a new website using responsive layouts and optimized user flow.

## Technology Stack
- HTML
- CSS
- JavaScript

## Features
- Responsive design
- Modern UI

## Setup and Run Instructions
1. Clone the repository  
2. Open index.html  
3. Run using browser or Live Server
